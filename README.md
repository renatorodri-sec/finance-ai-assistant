## 📌 Sobre o Projeto

O FinanceAI é um assistente virtual desenvolvido em Python com foco em educação financeira. O projeto foi criado para simular uma experiência de atendimento inteligente utilizando conceitos de Inteligência Artificial Generativa, Engenharia de Prompt e boas práticas de experiência do usuário (UX).

O assistente é capaz de responder dúvidas financeiras, realizar cálculos, explicar produtos financeiros e manter o contexto da conversa durante a interação.

---

## 🚀 Funcionalidades

- 💬 FAQ Inteligente
- 📈 Simulação de investimentos
- 💵 Cálculo de juros simples
- 💰 Cálculo de juros compostos
- 🏦 Explicação de produtos financeiros
- 🧠 Memória de contexto
- 🤖 Conversação com o usuário
