# Funcionalidades do FinanceAI

## Objetivo

O FinanceAI foi desenvolvido para auxiliar usuários na compreensão de conceitos de educação financeira por meio de um assistente virtual em Python.

---

# Funcionalidades Implementadas

## 💬 Chat Interativo

O usuário pode conversar com o assistente diretamente pelo terminal.

Exemplo:

Você:
```
O que são juros?
```

Resposta:

```
Juros são valores cobrados pelo uso de dinheiro emprestado ou ganhos sobre um investimento.
```

---

## 📚 Base de Conhecimento

O sistema possui uma base de conhecimento contendo informações sobre:

- Economia
- Investimentos
- Juros
- Cartão de crédito
- Orçamento
- Dívidas

---

## 🧠 Contexto da Conversa

Durante a execução, o histórico das mensagens é armazenado para manter o contexto da interação.

---

## 🧮 Cálculos Financeiros

O sistema possui funções para:

- Calcular Juros Simples
- Calcular Juros Compostos
- Simular Investimentos

---

## 🧹 Funções Utilitárias

O projeto inclui funções auxiliares para:

- Limpeza de texto
- Formatação monetária
- Validação de números
- Organização da interface

---

## ✅ Testes Automatizados

Todas as funcionalidades principais possuem testes automatizados utilizando Pytest.

Os módulos testados são:

- chatbot.py
- calculos.py
- conhecimento.py
- contexto.py
- utils.py
