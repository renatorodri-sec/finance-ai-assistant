# Roadmap do Projeto FinanceAI

## Objetivo

Este documento apresenta as funcionalidades planejadas para as próximas versões do FinanceAI.

---

# Versão 1.0 ✅

- Chat financeiro funcionando
- Base de conhecimento
- Cálculo de juros simples
- Cálculo de juros compostos
- Simulação de investimentos
- Histórico da conversa
- Testes automatizados
- Documentação do projeto

---

# Versão 1.1

- Melhorar a interface do terminal
- Adicionar mais perguntas e respostas
- Melhorar as mensagens do chatbot

---

# Versão 2.0

- Interface gráfica (GUI)
- Dashboard financeiro
- Gráficos de investimentos
- Cadastro de usuários

---

# Versão 3.0

- Banco de dados SQLite
- Histórico permanente das conversas
- Cadastro de despesas
- Cadastro de receitas

---

# Versão 4.0

- Integração com APIs financeiras
- Cotação de moedas
- Cotação de ações
- Informações sobre Tesouro Direto

---

# Versão 5.0

- Integração com Inteligência Artificial (LLM)
- Respostas mais inteligentes
- Sugestões personalizadas
- Assistente financeiro com memória

---

# Objetivo Final

Transformar o FinanceAI em um assistente financeiro completo, utilizando Inteligência Artificial, boas práticas de engenharia de software e recursos modernos da linguagem Python.
